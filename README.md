# Humidity Temperature Prediction using Linear Regression

A machine learning project that predicts **temperature** based on **humidity** using the Linear Regression algorithm. The project demonstrates the complete machine learning workflow, including data preprocessing, visualization, model training, prediction, and performance evaluation.

---

## Features

* Load and preprocess sensor data
* Perform exploratory data analysis (EDA)
* Visualize the relationship between humidity and temperature
* Train a Linear Regression model
* Predict temperature values
* Evaluate model performance using MSE and R² Score

---

## Technologies Used

* Python
* Google Colab / Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## Machine Learning Model

The project uses **Linear Regression**, a supervised machine learning algorithm that models the relationship between humidity (input feature) and temperature (target variable).

## Machine Learning Workflow

1. Load the dataset
2. Explore and analyze the data
3. Handle missing values
4. Visualize the relationship between humidity and temperature
5. Split the dataset into training and testing sets
6. Train the Linear Regression model
7. Predict temperature values
8. Evaluate the model using Mean Squared Error (MSE) and R² Score

---

## Results

The trained Linear Regression model predicts temperature values from humidity data. Model performance is evaluated using:

* Mean Squared Error (MSE)
* R² Score
